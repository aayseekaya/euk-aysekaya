#Local DB de laravel adında database oluşturulmalı
 
#Kurulum
 
```
composer install
```
```
php artisan key:generate
```
#Tabloların ve Dataların Oluşturulması

```
php artisan migrate:refresh --seed
```

#Oluşturulan dataların viewda gösterilmesi

```
php artisan serve
```
